A guide to getting started with XYZ-Code!

XYZ-Code aims to be a useful resource in helping you to learn how to build full-stack web applications from the ground up. It is suited for both beginners and veterans alike. This page aims to answer a few questions about XYZ-Code and provide a launch pad to to the best places to start on the site.

Why is XYZ-Code Different? Why should I use XYZ-Code?

XYZ-Code aims to take a Holistic approach to your development as developer. It doesn't focus on any one specific part of your development but rather aims to help you develop in a more balanced way accross all the 'core' areas. If you can progress well across all the core areas, then by the end of your development journey you will be in a good position to build a Full-Stack Web Application, utilising your new found knowledge to ensure you can approach all stages of the software development lifecyle well.

What are the Core Modules?

There are 6 core modules:

  • Programming Language - Understanding a programming language
  • Tools and Frameworks - What can you use to help better apply your programming language to a problem.
  • Algorithm Design - How to code correctly and efficiently
  • Design Patterns
  • SDLC - Understanding all the steps and skills required in the SDLC, from business feasability to Testing and Maintenance
